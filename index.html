Evaluater

A program that computes the precision and recall after receiving the datapoints and the predictions

View on GitHub Download .zip Download .tar.gz

Hello,

I’m a senior computer science student. I was working on a machine learning task to classify text, and I kept searching for a machine learning library to use. This task might seem too easy, but when it comes to a blind programmer who is new to machine learning, it wasn’t really easy to find a library with accessible documentations.

Until I found Vowpal Wabbit (VW). I started to use this library that has very good documentations, and it has been developed by Yahoo research in the past, and by Microsoft research currently.

Due to the fact that the library is a running project, it doesn’t have built in functions to calculate some values such as precision/recall/accuracy/etc.

Thus, I decided to code that by myself, and I thought to share that with the OpenSource community so that no one will have to re-implement that again.

It’s not a great project, but I believe it helps. Also, I sent to the Vowpal Wabbit developers asking to be added to contribute to the project even if I’ll update the documentation only.
